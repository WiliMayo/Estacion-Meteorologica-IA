# Estacion-Meteorologica-IA
Un proyecto de IoT que utiliza un ESP32 para medir temperatura/humedad (DHT), luminosidad (BH1750) y calidad del aire (MQ-135). Los datos se envían a la API de Gemini (Google AI) , que los analiza y genera recomendaciones prácticas. El sistema reacciona a las condiciones activando actuadores como un ventilador o LEDs.
